Ocean Pollution

This project proposes to use UAV images to detect economic activity and, as a result, estimate Ocean Pollution in a location. A Recurrent neural network is trained to learn various developmental parameters like rooftop type, source of lighting and proximity to water sources, Agriculture Area, Industrial Areas.
